Tiler
=====

A script creating bigger images by tiling one image to fit into given size. It supports a batch mode and fade-out borders.

Usage
-----
```
Usage: ./tiler.sh <width> <height> [--fadeout|-f] -- <files>```
e.g. $0 160x160 -f -- file1.png file2.png
```
